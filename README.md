# Гостевой дом «Абрикос»

Статический сайт гостевого дома «Абрикос», г. Ейск, ул. Советов, д. 12.

## Ветки

| Ветка | Назначение |
|-------|------------|
| **main** | Production — публичный сайт для [abrikos-yeisk.ru](https://abrikos-yeisk.ru) |
| **develop** | Разработка — полный репозиторий (сайт, прототип, документация) |

Разработка ведётся в **develop**. Перед релизом production-файлы переносятся в **main**.

Репозиторий: https://github.com/alfarius42/Abrikos
