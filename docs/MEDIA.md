# Медиа: куда класть фото и логотипы

Формат по умолчанию: **WebP**, соотношение **4:3**, ширина master ~1600 px.

## Брендинг

| Файл | Назначение |
|------|------------|
| `img/logo.png` | Логотип в шапке и favicon (вкладка, Apple Touch Icon) |
| `img/favicon-32.png` | Оптимизированная копия favicon 32×32 (генерируется из logo.png) |
| `img/archive/favicon.svg` | Архив: прежний упрощённый favicon |
| `img/logo-contrast.svg` | Предложенный контрастный знак в цветах сайта |

## Главная (`/`)

| Файл | Блок на странице |
|------|------------------|
| `img/home-block-1.webp` | «Добро пожаловать в Абрикос» |
| `img/home-block-2.webp` | «Ейск — жемчужина Азова» |
| `img/home-block-3.webp` | «Комфорт в каждой детали» |
| `img/home-block-4.webp` | «Для любой компании» |

## Номера — карточки и деталки

Для каждой категории (`1`, `2`, `3`, `4`, `5`, `6`, `apartments`):

| Файл | Где используется |
|------|------------------|
| `img/rooms/{id}-main.webp` | Карточка на `/rooms`, главное фото на `/rooms/{id}` |
| `img/rooms/{id}-gallery-1.webp` | Превью 1 на деталке |
| `img/rooms/{id}-gallery-2.webp` | Превью 2 на деталке |
| `img/rooms/{id}-gallery-3.webp` | Превью 3 на деталке |

Примеры:

- `img/rooms/1-main.webp`
- `img/rooms/apartments-main.webp`
- `img/rooms/apartments-gallery-1.webp`

## Страницы без фото-контента

- `/price` — только текст и таблица
- `/privacy` — только текст
- `/404` — без медиа

## OG / соцсети (опционально)

| Файл | Назначение |
|------|------------|
| `img/og-cover.jpg` | Превью для соцсетей и мессенджеров (1200×630) |

## Проверка после загрузки

1. Открыть `/rooms` — карточки подтянули локальные фото.
2. Открыть `/rooms/1` — галерея переключает 3 превью.
3. Открыть `/` — 4 блока показывают `home-block-*.webp`.
4. Проверить favicon во вкладке браузера.
