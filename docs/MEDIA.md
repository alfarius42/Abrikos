## Брендинг

| Файл | Назначение |
|------|------------|
| `img/logo.webp` | Логотип в шапке (основной) |

| `img/favicon-32.webp` | Favicon 32×32 для вкладки браузера |
| `img/hero.webp` | OG-обложка для соцсетей и мессенджеров |

## Номера — карточки и деталки

Для каждой категории (`1`, `2`, `3`, `4`, `5`, `6`, `apartments`, `kuban-house`):

| Файл | Где используется |
|------|------------------|
| `img/rooms/{id}-main.webp` | Карточка на `/rooms`, главное фото на `/rooms/{id}` |
| `img/rooms/{id}-gallery-2.webp` | Превью 2 на деталке |
| `img/rooms/{id}-gallery-3.webp` | Превью 3 на деталке |

У апартаментов дополнительно: `gallery-4`, `gallery-5`, `gallery-6`.

Примеры:

- `img/rooms/1-main.webp`
- `img/rooms/apartments-main.webp`
- `img/rooms/kuban-house-main.webp`

## Проверка после загрузки

1. Открыть `/rooms` — 8 карточек с локальными фото.
2. Открыть `/rooms/1` — галерея переключает превью.
3. Открыть `/territory` — галерея из 6 фото.
4. Открыть `/` — 4 блока показывают `home-block-*.webp`, карта с POI.
5. Проверить favicon во вкладке браузера.
